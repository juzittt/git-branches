# Ветки в Git 

Чтобы посмотреть все активные ветки в проекте, нужно вызвать команду   %BRANCH_NAME%
  bugfix/fix-branch
  feature/add-branch-info

# Ветки в Git 

Чтобы посмотреть все активные ветки в проекте, нужно вызвать команду   %BRANCH_NAME%
  bugfix/fix-branch
  feature/add-branch-info
* feature/diff
  master без аргументов. 

Для сравнения веток есть команда .
